Welcome To My Homepage

Perkenalkan nama saya Muhammad Fahri Putra Pratama biasa dipanggil fahri dari Universitas Negeri Malang. Saya lahir di Jawa Timur lebih tepatnya di Kota Malang pada tahun 2001. Semenjak dari kecil saya selalu mengambil pendidikan di kota tempat saya tinggal yaitu kota malang. Hal tersebut dikarenakan saya adalah dua bersaudara dan keluarga saya adalah keluarga yang sederhana jadi saya berpikir untuk tidak ke luar kota sebelum saya mampu menghasilkan pendapatan sendiri.

Saya adalah mahasiswa Teknik Informatika semester 7 atau angkatan 2019 di Universitas Negeri Malang. Saya memilih prodi TI ini karena selain prospek yang dimiliki oleh lulusan dari prodi ini sangat luas, saya juga dari awal sudah tertarik pada dunia teknologi. Ketertarikan saya pada dunia ini tentu saja karena hiburan yang disediakan oleh dunia teknologi contoh mudahnya adalah game. Pada saat memainkan beberapa game yang ada di internet saya merasa kagum dan penasaran mengenai bagaimana cara pihak developer membuat sebuah game yang begitu menarik minat banyak pemain, dari situlah awal mula rasa minat saya pada prodi ini muncul.

Tidak terlepas dari minat saya tadi, saya juga memiliki sebuah keinginan atau bisa disebut cita-cita untuk menjadi salah satu game developer ternama di dunia yang mampu membuat game terbaik seperti FIFA ataupun CS. Oleh karena itu, saya masuk ke dalam prodi ini dan untuk alasan kenapa saya memilih Akademi Fullstack Web Developer bukannya memilih MBKM yang berhubungan dengan pembuatan game, itu karena seorang programmer web yang mampu mengerjakan proyek secara fullstack banyak dicari oleh perusahaan, jadi ilmu fullstack ini nantinya akan saya gunakan sebagai cadangan atau alternatif apabila keinginan saya yang utama masih belum bisa tercapai.
